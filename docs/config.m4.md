# config.m4 分析

## 编译参数列表

```
--enable-debug-log          Enable swoole debug log, 默认为no
--enable-trace-log          Enable swoole trace log, 默认为no
--enable-scheduler-tick     
--enable-sockets            socket extension支持,默认为no
--enable-openssl            Use openssl
--enable-http2              Use http2.0?
--enable-swoole             默认yes
--enable-mysqlnd             Do you have mysqlnd?
--enable-debug              是否编译debug符号表  
```
